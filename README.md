# Hello World em Dart

**Estudante:** Kayky Kmita

## Objetivo

Configurar o ambiente de desenvolvimento e executar um programa simples utilizando a linguagem Dart, imprimindo a mensagem "Hello World" no terminal.

## Como executar

1. Certifique-se de ter o [Dart SDK](https://dart.dev/get-dart) instalado.
2. Clone o repositório:
   ```bash
   git clone <link-do-repositorio>
   ```
3. Navegue até a pasta do projeto:
   ```bash
   cd <nome-da-pasta>
   ```
4. Execute o programa:
   ```bash
   dart main.dart
   ```

## Saída esperada

```
Hello World
```
